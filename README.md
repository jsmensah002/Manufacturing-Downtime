Project Goal: Identify the key contributors to downtime and examine how operator actions and performance contribute to downtime occurrence and duration.

Tools: Power BI, Microsoft Excel

Key Insights:
- Overall production efficiency was 59.34%.
- Machine Adjustment was the largest contributor to downtime, totaling 332 minutes.
- Cola (CO-600) recorded the highest production runtime.
- Operator-related findings:
  - For Charlie and Dennis, the main downtime contributor was Machine Adjustment (An operator error).
  - For Dee, the main downtime contributor was Inventory Shortage.
  - For Mac, the main downtime contributor was Batch Change (An operator error).
