Project Goal: Identify the leading factors for downtime and examine how operators contribute to downtime occurrence and duration.

Tools: Power BI, Microsoft Excel. 

Dataset: Available upon request.

Key Insights:
- Overall production efficiency was 59.34%.
- Machine Adjustment was the largest contributor to downtime, totaling 332 minutes.
- Cola (CO-600) recorded the highest production runtime.
- Operator-related findings:
  - For Charlie and Dennis, the main downtime factor was Machine Adjustment (An operator error).
  - For Dee, downtime was mainly due to Inventory Shortage.
  - For Mac, the main downtime contributor was Batch Change (An operator error).
